Meus Projetos

Calculadora de IMC

A Calculadora de IMC é um projeto desenvolvido em React que utiliza Styled Components para estilização e useState para gerenciamento de estado. O usuário pode inserir peso e altura e o aplicativo calcula automaticamente o IMC, exibindo também a classificação correspondente. O código é organizado em componentes funcionais, reforçando conceitos de reatividade e reutilização. Este projeto demonstra habilidades em React, gerenciamento de estado e criação de interfaces interativas.

Página de evento tecnológico

A landing page foi desenvolvida utilizando HTML, SASS e JavaScript puro, explorando a biblioteca Math para cálculos e interações dinâmicas. A página conta com um cronômetro em tempo real, criado em JavaScript, que faz a contagem regressiva até o dia do evento. Com design moderno e responsivo, proporciona uma experiência de usuário fluida em diferentes dispositivos. Este projeto demonstra habilidades em manipulação do DOM, lógica em JavaScript, criação de interatividade e atenção ao design visual, reforçando fundamentos essenciais de front-end.
